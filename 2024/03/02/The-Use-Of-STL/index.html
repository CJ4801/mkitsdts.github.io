vector

vector is a data structure like array and actually it does like array. We can assess every data by []

Constructor Func

vectorv(n) create a vector of size n
vectorv(n,data) create a vector of size n and assign them to data

Member Func

The name of func function
begin(void),end(void) get the pointer of the first or last element

front(void) get the data of the first element

size(void) get the size of the vector

resize(int n) change the length of vector to n

pop_back() delete the last element

push_back(typename n) add an element in the end

at(typename* iterator) get the data that this pointer refer to

erase(iterator position) delete the data which iter pointer to
erase(iterator (beg,end)) delete the data from range beg to end
remove()

empty(void) judge the array whether has element. If it does,it will return 0
